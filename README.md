# AI-Integrated Jumanji Board Game

This project is an interactive board game that combines embedded systems, multiple sensors, BLE communication, and AI-generated challenges.  
The system uses two ESP32 boards, a Raspberry Pi, a BLE-enabled dice, ultrasonic sensors, hall-effect sensors, servos and LEDs to create a responsive, intelligent game experience.

---

## Overview

- Real-time player detection using ultrasonic sensors and LEDs  
- Embedded game mechanics implemented on ESP32 and Raspberry Pi  
- BLE-enabled dice using ESP32 Xiao  
- AI-generated challenges and trivia via Gemini API  
- Designed and built over three weeks in a four-person team

---

## Hardware Setup

**Main Components:**

- 2 × ESP32 boards  
- 1 × Raspberry Pi  
- 1 × ESP32 Xiao (BLE dice)  
- 4 × ultrasonic sensors  
- 20 × hall-effect sensors  
- 4 × servo motors  
- LEDs for feedback and player presence indication  
- Push buttons for player input
